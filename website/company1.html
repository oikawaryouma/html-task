～企業理念～



限られた人生の時間の中で、ビジネスを通して貴重な時間を共有させていただく。 限られた時間だからこそ、洗練されたサービスをもって、お客様に心地よさを感じていただきたい。

当社が掲げる最大にして唯一の行動指針は「お客様満足度」です。 お客様の期待を超えるサービスを実現し、思い出に残るサービスをご提供させていただくことで、公私にわたる一生涯のパートナーとなることが当社の目標です。

徹底的にサービスにこだわる、 おそらく世界初の本当の意味でのITサービス企業として、 お客様への最高のおもてなしを目指します。限られた人生の時間の中で、ビジネスを通して貴重な時間を共有させていただく。

企業沿革

年 起きたこと
1996年 (株)アイディーエス (IDS Corporation)設立
翻訳および文書管理事業を開始
1997年 インターネットを利用した翻訳システムを構築
1999年 東洋経済社発行ベンチャークラブ「21世紀を支える日本の企業家101人」に選出
2000年 日本工業新聞主催 「日本IT経営大賞」グランプリを受賞
インターネット接続型携帯電話認証サーバ「MITS（Mobile Internet Transaction Server）」を発売
2001年 「中小企業優秀新技術・新製品賞」を受賞
MITSが日本IBM社の「日本IBM Solution Excellence Award 2001優秀賞」を受賞
サイボウズ ケータイ向けブラウザフォン端末認証サーバ「MACK (Mobile Authentication for Cybozu Keitai)」を発売
2002年 日本初のアプライアンスサーバセット「携帯Bizメーラ(KBM)を発売」
2003年 携帯電話向け「GaMSS (Garoon Mobile Security System)」を発売
携帯電話端末認証サーバ「MITS」が特許を取得
2004年 ワンタイムパスワードソリューション「MITS OTP (Mobile Internet Transaction Server One Time Password)」を発売
(株)メディアドゥからのライセンスを受け、企業向けiモードパケット通信料削減システム「パケ割® Business Edition」を発売
2005年 (株)アイディーエス分社化
Web、ドキュメントおよび人材関連事業部門がグループ会社として独立
2006年 ネットバンキング向け「MITS-OTP」を発売
(株)オージス総研と「MITS-OTP」の販売代理店契約を締結
2007年 (株)エリアビイジャパンのリモートアクセスソリューション「SwanStor」を発売
Exchange/Notes携帯対応ソリューション「ConnectOne」 を発売
中野 貴志代表取締役に就任
Ajax型 Webメールソリューション 「Scalix」 を発売
2008年 ケアブレインズの「SugarCRM」を発売
Salesforce.comとのパートナー契約によりSaaS、PaaSビジネスを開始
社内ポータル構築事業を開始
投資信託販売業務では日本で初めて大日本印刷(株)、三菱UFJ信託銀行(株)向けに「デジタルペンシステム」を導入
社名を『International Digital Services』 に変更
福岡支社を設立
「Q転直Pa」を発売
2009年 Google社とのパートナー締結によりGoogleAppsビジネスを開始
2011年 システムエンジニアリング事業を開始
無料写真素材サイト「モデルピース」の立ち上げ
2013年 Amazon Web Servicesに特化したサニークラウド®事業を開始
業界初の無料請求代行サービス、環境構築・移行サービスを開始
Amazon Web Services (AWS)コンサルティングパートナーに認定
2014年 ロックオン社とEC-CUBE B2B認定販売代理店契約を締結
2015年 Amazon Web Services (AWS)アドバンスドコンサルティングパートナーに認定
ロックオン社のEC-CUBE B2Bの販売代理店（国内2社目）としてオリジナルサービスの販売を開始
2016年 「マーケティング・オートメーションサービス（Mautic）」の提供を開始
双日システムズ(株)、マジックソフトウェア・ジャパン(株)とデータ連携クラウドサービス「PolarisGate」を共同リリース
2017年 ベトナム ホーチミンシティに現地法人100%子会社IDS Vietnam Co., Ltdを設立
EC-CUBEゴールドパートナーに認定
2018年 ベトナムオフショアラボ開発のサービスを開始
2019年 Slack活用支援サービスを開始
リモートワーク導入支援サービスを開始
デジタルマーケティング支援サービスを開始
2020年 AWS エンドユーザコンピューティングコンピテンシーに認定
EC-CUBEプラチナパートナーに認定
「Sunny Pay FLAT」の提供を開始
2021年 IDS Vietnam増床
AWS「2021 APN AWS Top Engineers」に弊社のエンジニアが選出
AWS ISVパートナーの認定を取得
日本初のAmazon QuickSight「サービスデリバリープログラム」認定を取得
AWS「Well-Architectedパートナープログラム」認定を取得
